About this static build

This page is fully decoupled from Adobe Edge Delivery Services. There's no scripts/aem.js running, no reverse proxy, no live preview backend. It's HTML + CSS, served by any static file server.

Why use this

  • Quick visual sanity checks of block CSS without booting the AEM CLI.
  • Demos in environments with no network access.
  • Sharing with people who don't have the AEM toolchain installed.

Trade-offs

You lose the EDS pipeline: auto-blocking, the three-phase loader, dynamic block JS, and content authoring through AEM. The block JS files are not executed here, so blocks that rely on JS (forms, commerce, quiz, etc.) won't work. Pages that only use CSS-driven blocks (hero, cards, columns, default content) render fine.

Editing

Open static/index.html or static/about.html and edit. Refresh your browser. No build step.

Adding pages

Copy an existing file in static/, rename, change the content. Update the nav links in the header.
